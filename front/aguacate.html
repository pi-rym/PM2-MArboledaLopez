Sobre M2P

M2P o Module 2 Project, es una página de películas hecha para el módulo 2 de la etapa de Bootcamp de soyHenry. En esta página se pueden ver aplicados varios conocimientos, varios conceptos. Desde el inicio del desarrollo web con HTML, CSS y JavaScript básico, hasta la aplicación de JavaScript Avanzado. Es el reflejo mismo de todo el trabajo y aprendizaje de este módulo.
